# Elfo Javascript SDK

This is the Javascript (Typescript) SDK to interact with [elfo protocol programs](https://github.com/elfo-protocol/elfo-core)

## Installation

### yarn

```
yarn add @elfo-protocol/sdk
```

### npm

```
npm -i @elfo-protocol/sdk
```
