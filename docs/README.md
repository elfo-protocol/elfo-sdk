# Elfo Protocol JavaScript SDK

This is the Javascript SDK to interract with [elfo protocol programs](https://github.com/elfo-protocol/elfo)

## Installation

### yarn

```
yarn add @elfo-protocol/sdk
```

### npm

```
npm -i @elfo-protocol/sdk
```
