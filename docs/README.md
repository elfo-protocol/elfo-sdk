# Subrina Protocol JavaScript SDK

This is the Javascript SDK to interract with [subrina protocol programs](https://github.com/subrina-protocol/subrina)

## Installation

### yarn

```
yarn add @subrina-protocol/sdk
```

### npm

```
npm -i @subrina-protocol/sdk
```
